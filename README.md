Personal Portfolio Website

Make sure that node-gyp is proper globally
```shell
nvm install 23.6.0
npx node-gyp -v
```

Check for outdated dependencies
```shell
npm outdated
```

Run
```
npm run start
```

To post a new blog:
1. src > docs > blog
